= Gerrit 概览

Gerrit 是一个基于 WEB 端的 https://git-scm.com[Git] Code Review（代码审查）工具。这份说明简单介绍了 Gerrit，并描述了 Gerrit 是如何整合进一个典型的开发流程的。它同时描述了如何使用 Gerrit 去管理代码修改。

== Gerrit 是什么？

Gerrit 是一个轻量级 Code Review（代码审查）框架。无论是在开源项目中常见的仅有部分用户可以同意修改的项目，还是所有的贡献者都被充分信任的项目，Gerrit 都能很好的适应。
